## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

this will open the dev server. now you need to run the frames debugger along with next dev server.

open new terminal

```
frames
``` 

This will open the frames debugger.
